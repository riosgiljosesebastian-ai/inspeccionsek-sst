# Manual de Usuario para Inspección Sek SST

Este manual está diseñado para ayudar a los usuarios a navegar y utilizar eficazmente el sistema Inspección Sek SST.

## 1. Introducción
Inspección Sek SST es una herramienta diseñada para facilitar la gestión de la seguridad y salud en el trabajo, proporcionando informes y herramientas para la evaluación de riesgos.

## 2. Instalación
Para instalar la aplicación, siga estos pasos:
1. Clone el repositorio: `git clone https://github.com/riosgiljosesebastian-ai/inspeccionsek-sst.git`
2. Navegue al directorio: `cd inspeccionsek-sst`
3. Instale las dependencias: `npm install`

## 3. Uso
Para iniciar la aplicación, ejecute el siguiente comando:

```bash
npm start
```

## 4. Funciones Principales
- **Evaluación de riesgo:** Permite registrar y evaluar los riesgos asociados a diferentes tareas.
- **Informes:** Genera informes basados en las evaluaciones realizadas.

Para obtener asistencia adicional, consulte el [repositorio de issues](https://github.com/riosgiljosesebastian-ai/inspeccionsek-sst/issues) o contacte al soporte técnico.
